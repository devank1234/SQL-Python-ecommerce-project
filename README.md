# 🛒 E-Commerce Data Analysis using SQL & Python

## 📌 Project Overview

This project performs an end-to-end analysis of an e-commerce dataset using **MySQL, SQL, Python, Pandas, and Data Visualization**. The objective is to uncover valuable business insights related to customer behavior, sales performance, product demand, seller performance, and customer retention.

The project demonstrates the use of **Basic, Intermediate, and Advanced SQL concepts** along with Python for data extraction, analysis, and visualization.

---

## 🎯 Objectives

* Analyze customer purchasing behavior
* Identify top-performing product categories
* Measure seller performance and revenue contribution
* Evaluate customer retention patterns
* Track sales growth over time
* Generate actionable business insights for decision-making

---

## 📊 Business Problems Solved

### Basic SQL Analysis

1. List all unique cities where customers are located.
2. Count the number of orders placed in 2017.
3. Find the total sales per category.
4. Calculate the percentage of orders paid in installments.
5. Count the number of customers from each state.

---

### Intermediate SQL Analysis

1. Calculate the number of orders per month in 2018.
2. Find the average number of products per order grouped by customer city.
3. Calculate the percentage of total revenue contributed by each product category.
4. Analyze the relationship between product price and purchase frequency.
5. Calculate total revenue generated by each seller and rank them.

---

### Advanced SQL Analysis

1. Calculate moving averages of customer spending over time.
2. Compute cumulative monthly sales for each year.
3. Calculate Year-over-Year (YoY) sales growth.
4. Measure customer retention within 6 months of first purchase.
5. Identify the top 3 highest-spending customers for each year.

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/devank1234/SQL-Python-ecommerce-project.git
```

2. Install required libraries

```bash
pip install pandas mysql-connector-python matplotlib
```

3. Create a MySQL database

```sql
CREATE DATABASE ecommerce;
```

4. Update MySQL credentials in `csv_to_sql.py`

5. Run the data import script

```bash
python csv_to_sql.py
```

6. Open the Jupyter Notebook and execute the analysis.
